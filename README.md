# Finepro's Project Management

Gain access to powerful tools that will help you manage your projects with ease, collaborate seamlessly with your team, and unlock the full potential of every task. Start taking your projects to the next level and achieve more, faster, and smarter.

## Installation

- Clone sources

- Install packages

```bash
  npm install
```

- Create file _.env_

```bash
  cp .env
```

Add environment variables `PORT` `APP_ENV` `BASE_URL` to .env file

- Run a test

```bash
  npm run dev
```

## Tech Stack

- React (TypeScript)
- TailwindCSS
- Zustand
